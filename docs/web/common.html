# http method

GET 的請求應用於從後端取得資料。

POST 的請求用於提交資料至後端。

PATCH 的請求用於套用資料至後端修改。

DELETE 的請求用於刪除後端指定資料

2XX 系列 2XX 系列代表 server 端有成功接受到請求。

200 OK 請求已成功，且馬上回傳資料。

3XX 系列 3XX 系列代表重新導向，這類的 status code 需要 client 端進行更進一步的操作才能完成請求，譬如登入後會重新導向至首頁之類的操作。

4XX 系列 4XX 系列代表 client 端有錯誤。

404 Not Found 這個 status code 想必大家應該都看過吧 XD 只要頁面不存在通常都會顯示這段文字

5XX 系列 5XX 系列代表 server 端在處理請求的過程中有錯誤，看到這類代表不用檢查自己程式碼是否有寫錯，趕快跟後端工程師反應吧 XD
